# Plano de Testes — Lia Simplis

## Tipos de teste

### 1. Aceitação
Validar que o app atende os requisitos funcionais do ponto de vista do usuário.

### 2. Desempenho
Medir tempo de resposta e estabilidade em operações repetitivas.

## Ambiente

- **App**: Lia Simplis Flutter (web)
- **Dispositivo**: navegador Chrome
- **Build**: `flutter run -d web-server --web-port=8080`
